Welcome to i3ctl's documentation!
===============================

**i3ctl** is a comprehensive command line utility for managing i3 window manager settings.

Features:
---------

* Brightness control
* Volume control
* Wallpaper management
* Keyboard layout management
* i3 config management
* Startup application management
* Power management
* Network management
